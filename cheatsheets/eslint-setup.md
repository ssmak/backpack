# ESLint Setup

```bash
# Install ESLint locally
npm i -D eslint

# Init ESlint
npx eslint --init

# For Typescript and React project. Add below line to package.json
# "lint": "eslint \"**/*.{ts,tsx}\""

# Lint all the pattern matched files
yarn lint
```
